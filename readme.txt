== Description ==


My Timeline Blog is the WordPress Plugin that creates Responsive Timeline blog Page for you, using this plugin users can create own responsive Vertical timelines, 
user can go the specific year of blog by clicking the top year tab section.



Shortcode [mtb-blog]to create My Timeline Blog on Page or Post.


= My Timeline Blog Features =

* Unlimited Free timelines blog
* Responsive UI that looks cool on every single device
* Beautiful UI of the vertical timeline
* Add dates for each Post of the timeline
* year tab on top of timeline blog


HOW TO USE IT
[mtb-blog]

post the [mtb-blog] on your page or post to display the responsive timeline blog

== Installation ==

Installation of this plugin is pretty easy.

1. Upload the extracted plugin folder to the `/wp-content/plugins/` directory of your WordPress installation, or upload it directly from your plugins section in the WordPress admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it!

== Frequently Asked Questions ==

= Getting Page Not Found on Read More button =

Goto Settings -> Permalinks -> Select "Post name" Name -> Click Save Changes

If Post name is already selected just click "Save Changes"

= How can i use this plugin? =

1. To show timeline on a Page or Post, Click on Timeline Blog menu and Add New.

Create a ticket in support for further queries.


= Why my Timeline Blog is showing on Right Side of page with scroll bar? =

Timeline works with 100% space on page, when you will make your container 100% with css, the timeline will appear properly.
